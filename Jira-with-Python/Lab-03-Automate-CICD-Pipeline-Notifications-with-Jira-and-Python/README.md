# Lab 03 - Automate CI/CD Pipeline Notifications with Jira and Python

## Objective

Implement a Jenkins CI/CD pipeline and integrate it with Jira using Python automation scripts.

## Tools Used

- Jenkins
- Python 3
- Jira Cloud
- Jira REST API
- Git
- Requests
- Python Dotenv

## Features

- Jenkins pipeline execution
- Jenkins build monitoring
- Pipeline stage monitoring
- Automated Jira issue creation
- Automated Jira issue updates
- CI/CD status notifications

## Project Structure

- Jenkins pipeline configuration
- Jenkins monitoring scripts
- Jira integration scripts
- Automated notification workflow

## Outcome

Successfully integrated Jenkins CI/CD pipelines with Jira using Python automation and REST APIs.
