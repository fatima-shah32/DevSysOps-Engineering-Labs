# Lab 07 - Governance and Compliance Checklist Automation

## Objective

Automate Governance, Risk, and Compliance (GRC) checks using Python and generate compliance reports with automated issue tracking.

## Tools Used

* Python 3
* Linux
* JSON
* Logging
* Git

## Features

* Automated GRC compliance checks
* Password policy auditing
* SSH security validation
* File permission auditing
* Service monitoring
* Log integrity verification
* Automated issue creation
* Compliance reporting

## Project Structure

```text
Lab-07-Governance-and-Compliance-Checklist-Automation/
├── checks/
│   └── grc_checker.py
├── reports/
│   └── issue_tracker.py
├── config/
│   └── grc_checks.json
├── logs/
├── grc_automation_main.py
├── README.md
└── .gitignore
```

## Run

```bash
python3 checks/grc_checker.py
python3 reports/issue_tracker.py
python3 grc_automation_main.py
```

## Outcome

Successfully automated compliance validation, generated compliance reports, and created issue records for failed GRC controls.
