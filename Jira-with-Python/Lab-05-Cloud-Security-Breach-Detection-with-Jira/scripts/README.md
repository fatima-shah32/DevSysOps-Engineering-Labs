# Lab 05 - Cloud Security Breach Detection with Jira

## Objective

Simulate cloud storage security breaches, detect suspicious activities using Python, and generate Jira security incidents automatically.

## Tools Used

- Python 3
- JSON
- Jira Cloud
- Requests
- Python Dotenv

## Project Structure

Lab-05-Cloud-Security-Breach-Detection-with-Jira/
├── config/
├── logs/
├── scripts/
├── README.md
├── requirements.txt
└── .gitignore

## Workflow

1. Simulate cloud storage access logs
2. Detect security breaches
3. Generate security incident reports
4. Create Jira security tickets

## Scripts

- simulate_breach.py
- breach_detector.py
- jira_integration.py

## Outcome

Successfully simulated cloud security incidents, detected suspicious activities, categorized breaches by severity, and integrated incident reporting with Jira.
